# Caramelo Clicker 🎮

Um jogo no estilo clicker protagonizado pelo personagem mais querido do Brasil: o cachorro caramelo.

O objetivo do jogador é ajudar o Caramelo a transformar uma pequena muda em uma gigantesca Árvore Lendária através de cliques, melhorias, desbloqueio de skins e evolução da planta.

## 📚 Sobre o Projeto

O projeto consiste no desenvolvimento de um jogo clicker desenvolvido para a disciplina de Desenvolvimento Web da UNIFEBE.

Durante a partida, o jogador controla o Caramelo, que utiliza diferentes equipamentos para regar uma planta e gerar folhas, a moeda principal do jogo.

Conforme o jogador acumula folhas, é possível adquirir melhorias, aumentar o poder de clique e desbloquear novas skins para o personagem.

A cada fase concluída, a planta evolui até atingir sua forma final: a lendária Árvore do Caramelo.

---

## 🎮 Gameplay

O jogo segue a mecânica clássica de clicker games:

🍃 Clique no Caramelo para gerar folhas.

🪴 Encha o medidor de crescimento para evoluir a planta.

🛒 Compre upgrades para aumentar a quantidade de folhas obtidas por clique.

🔥 Ative o bônus de combo clicando continuamente.

👕 Desbloqueie novas skins conforme acumula folhas.

🌳 Evolua a planta através de cinco estágios de crescimento.

✨ Alcance a Árvore Lendária para concluir o jogo.

---

## 🌳 Evolução da Planta

A planta evolui através de cinco fases:

🌱 Muda

🌿 Arbusto

🌳 Árvore Jovem

🌲 Árvore Adulta

✨ Árvore Lendária

Cada evolução exige que o jogador complete o medidor de crescimento utilizando as folhas obtidas durante a partida.

---

## 👕 Sistema de Skins

Conforme o jogador progride, novas versões do Caramelo são desbloqueadas:

🐕 Normal

🦴 Motivado

🚿 Wapp Milagroso

🛸 Espacial

🎇 Lendário

Após desbloqueadas, as skins podem ser selecionadas livremente através do menu do jogo.

---

## 🛒 Upgrades

O jogador pode adquirir melhorias que aumentam a quantidade de folhas recebidas por clique.

### 🦴 Petisco Motivacional

Aumenta o ganho de folhas por clique.

### 🚿 Wapp Milagroso

Permite uma evolução mais rápida da planta.

### 🛸 Arminha de Água Espacial

Grande aumento na produção de folhas.

### 🌧️ Capa de Chuva do Caos

Uma das melhorias mais poderosas do jogo.

Os upgrades podem ser comprados diversas vezes, aumentando progressivamente seu custo.
